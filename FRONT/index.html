Informe: Sistema de Gestión de Recolección de Hardware

Plataforma propuesta para registrar solicitudes, clasificar equipos y generar reportes de reciclaje y reutilización.

Resumen ejecutivo

  • Registro de usuarios y solicitudes.
  • Clasificación de hardware recolectado.
  • Generación de reportes resumidos.
  • Arquitectura ligera: SQLite + API REST.

Sobre el proyecto

HardwareRevive busca reducir el impacto ambiental mediante la recolección, clasificación y reutilización de componentes electrónicos.

Alcance y funcionalidades

  • CRUD de usuarios y solicitudes.
  • Inventario y clasificación de hardware.
  • Reportes con estadísticas resumidas.

Arquitectura del sistema

La solución está basada en tres capas: base de datos SQLite, API REST (Flask o Node.js), y una interfaz web CRUD.

users

id (PK), nombre, email, rol

requests

id (PK), user_id, direccion, estado

hardwares

id, tipo, marca, modelo, estado
